This endpoint returns the list of all email accounts in your Reply account.
Throttling:

Field	Type	Description

emailAddress	string	Email account email address.

HEADERS
x-api-key

{{apiKey}}

POSTUpdate Email Account
https://api.reply.io/v1/EmailAccounts
Description:

This endpoint allows you to modify the details of an existing Reply account used for sending emails.
Notes:

    You can only update email accounts by using the "other provider" setup.

Throttling:

    There is no throttling for this endpoint.

Request Body Data:
Field	Type	Description	Required
Email	string	The email address used for sending emails.	Yes
Daily Limit	integer	The maximum number of emails you can send per day	Yes
SenderName	string	The name that will be displayed as the sender in email recipients' inboxes	Yes
SmtpHost	string	The hostname or IP address of the Simple Mail Transfer Protocol (SMTP) server used for sending emails.	Yes
SmtpPort	integer	The port number used for communication with the SMTP server (typically 465 for SMTP with SSL/TLS encryption).	Yes
SmtpPassword	string	The password for the email address used for sending emails	Yes
SmtpSsl	boolean	This option controls whether SSL is used for the SMTP connection	No
ImapHost	string	The hostname or IP address of the Internet Message Access Protocol (IMAP) server used for receiving emails.	Yes
ImapPort	integer	The port number used for communication with the IMAP server (typically 993 for IMAP with SSL/TLS encryption).	Yes
ImapPassword	string	The password for the email address used for receiving emails	Yes
ImapSsl	boolean	This option controls whether SSL is used for the IMAP connection	No
Response Body Data:
Field	Type	Description
id	integer	Updated Email account ID.
HEADERS
x-api-key

{{apiKey}}

Your API Key from settings
Bodyraw (json)
json
