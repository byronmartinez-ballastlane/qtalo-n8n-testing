POSTSet Schedule as default by ID
{{url}}/v2/schedules/{{id}}/set-default
Description:

Sets an existing schedule as the default schedule. Once set, this schedule will automatically be applied to new sequences or campaigns unless another schedule is explicitly specified.
Throttling:

    There is no throttling for this endpoint

Request Body Data

This endpoint does not require a request body.
All necessary information is passed via headers.
Required Headers
Header	Type	Required	Description
x-api-key	string	Yes	Your personal API key.
Content-Type	string	Yes	Must be set to application/json
Response Body Data

No response body (returns status code only).
Status Codes
Status Code	Description
200	Schedule set as default successfully
401	Unauthorized access
404	Schedule not found
500	Internal server error
Notes

    Setting a schedule as default may unset the previous default schedule.

HEADERS
x-api-key

{{apikey}}
PARAMS
{{id}}

Unique ID of schedule
Example Request
Set Schedule as default by ID
curl

curl --location -g --request POST '{{url}}/v2/schedules/{{id}}/set-default' \
